源速:配合我的上传工具yssync 使您的工作事半功倍!

安装yssync

yssync是一个命令行工具，使用npm安装

npm install yssync -g

yssync需要先安装node and npm

源速:让我们来使用yssync命令吧!

首先请打开命令行窗口

初始化目录

cd workdirectory   
yssync init -s <safecode> -n <spacename>

yssync 在当前目录下生成 .yssync 配置文件,您可以直接修改文件里面的参数

上传所有文件

yssync upload

了解yssync更多功能

yssync -h

	Usage: yssync [options] [command]
	Commands:

	init     初始化当前路径为源速同步目录
	upload   上传当前目录文件到源速

	Options:

	-h, --help                 output usage information
	-V, --version              output the version number
	-s --safecode      设置API安全码
	-n --namespace     设置空间名称
	-p --prefixPath    设置前缀路径
	-i --isCover      是否覆盖同名文件
	-w --workerLimit   并行上传文件数量
